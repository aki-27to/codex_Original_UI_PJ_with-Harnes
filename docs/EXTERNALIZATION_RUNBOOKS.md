# Externalization Runbooks

Updated: 2026-04-12

## Operator Runbook

operator 向けには、public-safe proof と current truth を先に見せます。

## Human Evaluator Runbook

human evaluator には release-ready evidence chain と residual risk を渡します。

## External Auditor Runbook

external auditor には redacted bundle と traceable contract ref を渡し、raw local log 依存にしません。

## Deployment Evidence Runbook

deployment / ship-no-ship 判断には governance_public bundle と signoff summary を使います。
