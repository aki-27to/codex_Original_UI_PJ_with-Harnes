# GLOSSARY

Authority role: `navigation / terminology only`  
Authority registry: `authority-registry.v1`

この文書は、repo 内で頻出する用語を短く揃えるための glossary です。

- governed harness
  - 固定された authority と proof contract の内側で worker を動かす実行基盤
- companion surface
  - core truth ではないが、隣接 product / workflow として併置される面
- current surface
  - logs/current/ にある fixed five summary files を中心とした operator 向け最新 surface
- signoff bundle
  - release / signoff 判断のために束ねられた repo-safe proof bundle
- runtime/
  - transient local material を置く場所
- output/
  - public-safe report、proof、readiness artifact を置く場所
