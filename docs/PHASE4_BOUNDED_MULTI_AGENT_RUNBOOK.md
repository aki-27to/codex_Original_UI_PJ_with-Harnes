# PHASE4_BOUNDED_MULTI_AGENT_RUNBOOK

Updated: 2026-04-12

## 1) Purpose

Phase 4 は bounded multi-agent collaboration を、specialist dispatch + evidence bundle + release honesty の範囲で完成させます。

## 2) Collaboration Rule

- parent は dispatch / aggregation
- child は material execution
- reviewer/tester は independent assurance

## 3) Required Evidence

- child outcome
- reviewer/tester evidence
- signoff bundle
- residual risk

## 4) Boundedness

multi-agent だからといって独自 orchestration sprawl を増やしてはいけません。
