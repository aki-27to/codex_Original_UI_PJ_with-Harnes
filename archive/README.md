# Archive

This directory holds legacy or reference-only material that is not part of the active harness runtime surface.

Current contents:
- `doc_old/`: retired docs kept for historical reference
- `TESTFolder/`: archived example/static website work
- `outputs/`: older manual render outputs
- `installers/`: installer drops and external binary packages kept outside the active root

Active runtime surfaces remain at the repository root:
- `docs/`
- `scripts/`
- `tools/`
- `web/`
- `logs/`
- `output/`
